FrozenDict¶

class frozendict.frozen_dict.FrozenDict(*args, **kwargs)¶

Bases: Mapping[frozendict.frozen_dict.KT, frozendict.frozen_dict.VT_co]

An immutable dictionary that implements the “py:class:~typing.Mapping protocol and other dict methods.

Instantiate a FrozenDict.

Parameters

  • value – A mapping from which to create a FrozenDict.

  • homogeneous_type – Option to check that all types in the dictionary are homogeneous, all keys should have the same type and all values should have the same type.

  • remove_none_values – Option to remove any None value from the given mapping.

  • no_copy – Option to disable the copy of the given mapping. Ex: frozendict({k1:v1, k2:v2}, no_copy=True) will create a safe and immutable object without copying as there are no references of the given value.

  • kwargs – You can use kwargs to instantiate a FrozenDict. Ex: FrozenDict(k1:v1, k2:v2)

update(mapping)¶

Returns a new immutable dict that contains the key:value elements from this dict, updated with the key:value elements from the given mapping.

Note that this method is not compatible with the dict.update method, which updates the dictionary in-place (destructively), while this method is non-destructive.

Parameters

mapping (Mapping) – Required. Either another dictionary object or an iterable of key:value pairs (iterables of length two). If keyword arguments are specified, the dictionary is then updated with those key:value pairs.

Returns

The new frozen dictionary.

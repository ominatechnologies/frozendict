NoCopyFrozenDict¶

class frozendict.no_copy_frozen_dict.NoCopyFrozenDict(*args, **kwargs)¶

Bases: frozendict.frozen_dict.FrozenDict[frozendict.no_copy_frozen_dict.KT, frozendict.no_copy_frozen_dict.VT_co]

Instantiate a FrozenDict without making a copy of the given dictionary.

WARNING: By definition it is not immutable but will perform better. A good usage would be when you instantiate a frozendict inline. Ex: NoCopyFrozenDict({k1:v1, k2:v2}) is safe and immutable

Parameters

  • value – A mapping from which to create a FrozenDict.

  • homogeneous_type – Option to check that all types in the dictionary are homogeneous, all keys should have the same type and all values should have the same type.

  • remove_none_values – Option to remove all None value from the given mapping.

  • no_copy – Option to enable the copy of the given mapping.

  • kwargs – You can use kwargs to instantiate a FrozenDict. Ex: FrozenDict(k1:v1, k2:v2)

v1.2 - 2020-04-15¶

  • Add the ‘no_copy’ parameter to control whether the given dictionary is copied in the ‘FrozenDict’ constructor, which is False by default.

  • Add the ‘NoCopyFrozenDict’ class for which the ‘no_copy’ parameter is True by default and which can be imported as a drop-in replacement using: from frozendict import NoCopyFrozenDict as FrozenDict.

  • Various fixes.
